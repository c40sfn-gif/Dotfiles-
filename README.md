# ⚡ Sway Dotfiles
# BY FR4N
Um conjunto de **dotfiles modernos e modulares** para configurar rapidamente um ambiente Linux produtivo, bonito e funcional.

Este projeto permite que qualquer pessoa — mesmo sem experiência avançada — consiga instalar e utilizar um setup completo de terminal e desktop com poucos comandos.

---

## 🧠 O que são Dotfiles?

Dotfiles são arquivos de configuração usados por programas no Linux e Unix.

Eles permitem:

* personalizar aparência do sistema
* configurar terminal e shell
* automatizar instalação de ferramentas
* recriar seu ambiente em outra máquina rapidamente

## 🚀 Instalação rápida

Clone o repositório:

```
git clone https://github.com/c40sfn-gif/Dotfiles-.git
cd Dotfiles-
```

Execute o instalador:

```
bash install.sh
```
